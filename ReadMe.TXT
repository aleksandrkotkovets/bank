
Первый пункт!
  Разместите папку с пиложением на диске E Вашего компьютера. 
  Для запуска программы откройте консоль: 
 	1. Нажмите сочитание клавиш Win+R;
	2. В появившемся окне введите "cmd" и нажмите "Enter".
  В консоли введите: java -jar E:\kursWork\Server\out\artifacts\Server_jar\Server.jar
  Нажмите "Enter".
  Сервер запущен.

Второй пункт!
   Откройте вторую консоль. 
   В консоли введите: java -jar E:\kursWork\Client\out\artifacts\Client_jar\Client.jar
   Нажмите "Enter"

Программа запущена!

Чтобы открыть файл с результатом подсчетов, необходимо перейти в папку по следующему пути:
	"E:\kursWork\Client\src\\kursWork\record.txt"

Для входа под администратором необходимо ввести пароль и логин:
	Логин: 1
	Пароль: 1
Ввод логина и пароля без пробелов  !!!!!
